# 04-2 Python的文字-切割文字

### Python的文字資料可以使用「中括號」選擇文字段落.  

以下是一個使用文字資料的例子:

## [範例-1] 畫面示例
![GitHub Logo](/images/04-2-1.jpg)

## [範例-1] 程式
```python
#---------------------------
# 切割文字範例
#---------------------------
a='國立臺北商業大學'

#切割前2個字
b=a[0:2]    

#切割後2個字
k=len(a)    
c=a[k-2:k]

#印出結果
print(a)
print(b)
print(c)
```

## [範例-1] 執行結果
```
國立臺北商業大學
國立
大學
```
