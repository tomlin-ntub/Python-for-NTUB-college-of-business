# 06-5 自訂函式

### 經常重覆使用的方法可以寫成類別或函式.


## [範例-1] 畫面示例
![GitHub Logo](/images/06a-5-1.jpg)

## [範例-1] 程式
```python
#-----------------------------
# 一個計算傭金的函式
# 傳入:銷售金額
# 回傳:傭金
#-----------------------------
def commission(total):
    rate=0.0
    
    if total>20000:
        rate=0.15
    elif total>10000:
        rate=0.1
    else:
        rate=0.08
        
    return int(total*rate)
#-----------------------------    



#-----------------------------
# 主程式
#-----------------------------
tot=[25000, 36000, 12000, 9500, 45000]

for t in tot:
    print('銷售{}元, 傭金{}元'.format(t, commission(t)))   
```


## [範例-1] 執行結果
```
銷售25000元, 傭金3750元
銷售36000元, 傭金5400元
銷售12000元, 傭金1200元
銷售9500元, 傭金760元
銷售45000元, 傭金6750元
```



## [範例-2] 程式
```python
#-----------------------------
# 一個計算傭金的函式
# 傳入:銷售金額, 職級
# 回傳:傭金 
#-----------------------------
def commission(total, rank):
    rate=0.0
    
    if rank=='A':
        if total>20000:
            rate=0.15
        elif total>10000:
            rate=0.1
        else:
            rate=0.08
    else:
        if total>20000:
            rate=0.12
        elif total>10000:
            rate=0.09
        else:
            rate=0.05        
        
    return int(total*rate)   
#-----------------------------



#-----------------------------
# 主程式
#-----------------------------
tot=[(25000, 'A'), (36000, 'B'), (12000, 'B'), (9500, 'A'), (45000, 'A')]

for t, r in tot:
    print('銷售{}元, 職級{}, 傭金{}元'.format(t, r,  commission(t, r))) 
#-----------------------------    
```


## [範例-2] 執行結果
```
銷售25000元, 職級A, 傭金3750元
銷售36000元, 職級B, 傭金4320元
銷售12000元, 職級B, 傭金1080元
銷售9500元, 職級A, 傭金760元
銷售45000元, 職級A, 傭金6750元
```
