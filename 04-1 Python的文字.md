# 04-1 Python的文字

### Python的「文字」型態資料必須用「單引號」或「雙引號」框在兩邊; 它們必須配對, 而且兩者使用一致的符號. 

以下是一個使用文字資料的例子:

## [範例-1] 畫面示例
![GitHub Logo](/images/04-1-1.jpg)

## [範例-1] 程式
```python
#---------------------------
# 使用文字
#---------------------------
a='國立臺北商業大學'
b="財經學院"

print('Hello')
print(a)
print(b)
```

## [範例-1] 執行結果
```
Hello
國立臺北商業大學
財經學院
```
